# DevOps Step 1 - Git Workflow

## Goal
Learn basic Git workflow for development and collaboration.

---

## Basic Commands
git init  
git status  
git add .  
git commit -m "message"  
git push  
git pull  
git log  

---

## Branching
main = stable code  
feature branch = new work  

Create branch:
git checkout -b feature/test  

Switch branch:
git checkout main  

---

## Workflow Steps
1. Create branch
2. Make changes
3. git add .
4. git commit -m "update"
5. git push origin feature/test
6. Create Pull Request (GitHub)
7. Review code
8. Merge to main

---

## Merge
- Merge PR into main branch
- Delete feature branch (optional)

---

## Full Flow
main → branch → code → commit → push → PR → review → merge

---

## Outcome
- Git basics learned
- Branching understood
- PR workflow practiced   
- Team collaboration ready
